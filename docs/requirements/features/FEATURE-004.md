# FEATURE-004 저장소 관리와 진입 흐름

## 상위 Epic

- [EPIC-001 학습용 기보 저장 및 분석](../epics/EPIC-001.md)

## 브랜치명

- `feature/FEATURE-004-저장소-관리`

## 목적

- 사용자가 여러 저장소를 만들고, 원하는 저장소로 들어가 학습 작업을 시작할 수 있어야 한다.

## 동작 개요

- 저장소 목록은 생성 순서대로 보여준다.
- 사용자는 저장소를 생성하고 이름을 정할 수 있다.
- 사용자는 저장소를 선택해 진입할 수 있다.
- 사용자는 저장소를 삭제할 수 있고, 삭제 전에는 반드시 확인 과정을 거친다.
- 저장소 진입 시 기본 화면은 FEATURE-002에서 만든 보드 중심 뷰를 기반으로 한 저장소 기본 보드뷰다.
