# FEATURE-006 분석 보드와 저장소 통계 조회

## 상위 Epic

- [EPIC-001 학습용 기보 저장 및 분석](../epics/EPIC-001.md)

## 브랜치명

- `feature/FEATURE-006-분석-보드-통계`

## 목적

- 사용자가 저장소 단위로 기보를 분석하고, 특정 포지션 다음 수 후보와 승률을 확인할 수 있어야 한다.

## 동작 개요

- 저장소 진입 시 기본 화면은 분석 보드 뷰다.
- 분석 보드 뷰는 보드를 기준으로 포지션을 만들고 통계를 조회하는 작업 화면이다.
- 분석용 포지션은 보드에서 수를 직접 두어 만들 수 있다.
- 분석 보드에서도 되돌리기와 다시하기를 지원한다.
- 통계는 저장소 안의 정식 저장 기보를 기준으로 집계한다.
- 초기 통계는 특정 포지션 다음에 많이 둔 수와 각 수의 백승/무승부/흑승 비율을 보여준다.
