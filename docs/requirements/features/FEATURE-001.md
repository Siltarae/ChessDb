# FEATURE-001 체스 규칙 및 합법 수 판정 엔진

## 상위 Epic

- [EPIC-001 학습용 기보 저장 및 분석](../epics/EPIC-001.md)

## 브랜치명

- `feature/FEATURE-001-체스-엔진`

## 목적

- 기물별 이동 규칙, 체크 판정, 캐슬링, 프로모션 등 체스 도메인의 핵심 로직과 합법 수 판정 기능을 구현한다.

## 동작 개요

- 보드의 현재 상태를 표현하는 핵심 데이터 구조를 정의한다.
- 현재 보드 상태를 기반으로 각 기물이 이동할 수 있는 모든 합법 수를 계산한다.
- 킹이 공격받는 체크 상태를 정확히 판정한다.
- 체크 상태에서는 킹을 보호하거나 피하는 수만 합법 수로 필터링한다.
- 캐슬링, 앙파상, 프로모션 등 특수 이동 규칙을 처리한다.
- 체크메이트, 스테일메이트, 50수 규칙, 3회 동형 반복 등 게임 종료 조건을 판정한다.
