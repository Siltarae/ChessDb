# FEATURE-005 기보 목록과 상세 조회

## 상위 Epic

- [EPIC-001 학습용 기보 저장 및 분석](../epics/EPIC-001.md)

## 브랜치명

- `feature/FEATURE-005-기보-상세`

## 목적

- 사용자가 저장된 기보를 목록에서 찾아 열고, 상세 화면에서 수순과 코멘트를 학습용으로 다시 볼 수 있어야 한다.

## 동작 개요

- 저장된 기보 목록은 날짜순으로 보여준다.
- 목록에서 특정 기보를 선택하면 기보 상세 뷰로 이동한다.
- 상세 뷰에서는 보드, 수순 목록, 코멘트, 착수 평가 기호를 함께 보여준다.
- 상세 뷰에서 수순을 선택하면 해당 시점의 보드 상태로 이동한다.
- 정식 저장된 기보의 수순은 수정하지 않지만, 코멘트와 착수 평가 기호는 추가하거나 수정할 수 있다.
- 코멘트와 착수 평가 기호 수정은 즉시 자동 저장된다.
