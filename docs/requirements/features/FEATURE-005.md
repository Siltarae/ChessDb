# FEATURE-005 기보 목록과 상세 편집

## 상위 Epic

- [EPIC-001 학습용 기보 저장 및 분석](../epics/EPIC-001.md)

## 브랜치명

- `feature/FEATURE-005-기보-상세`

## 목적

- 사용자가 저장된 기보를 목록에서 찾아 열고, 상세 화면에서 수순, 코멘트, 착수 평가 기호, 기보 정보를 확인하고 수정할 수 있어야 한다.

## 동작 개요

- 저장된 기보 목록은 날짜순으로 보여준다.
- 목록에서 특정 기보를 선택하면 기보 상세 편집 뷰로 이동한다.
- 상세 편집 뷰에서는 보드, 수순 목록, 코멘트, 착수 평가 기호, 기보 정보를 함께 보여준다.
- 상세 뷰에서 수순을 선택하면 해당 시점의 보드 상태로 이동한다.
- 정식 저장된 기보의 코멘트, 착수 평가 기호, 기보 정보는 수정할 수 있다.
- 정식 저장된 기보의 수순도 수정할 수 있어야 하며, 수순 수정은 이후 상태 재계산과 반수별 메타데이터 정리를 함께 다룬다.
- 코멘트와 착수 평가 기호 수정은 즉시 자동 저장된다.
- 새 기보 입력 화면과 상세 편집 화면은 장기적으로 공통 기보 편집 경험을 공유할 수 있다.

## 메모

- 상대 이름은 현재 범위에서 다루지 않는다.
- 기보 제목과 즐겨찾기는 후순위 범위다.
- 수순 수정은 단순 표시 변경이 아니라 이후 수순 재계산, 코멘트/어노테이션 반수 연결, 저장 API 갱신을 포함하는 별도 구현 단위로 다룬다.
- 저장소 기본 보드뷰의 저장 버튼이 기존 기보를 업데이트하는 기능은 별도 Task로 분리한다.
- 저장소 기본 보드뷰의 복원 동작은 영구 저장이 아니라 로컬 작업 상태를 원본 기준으로 되돌리는 별도 Task로 분리한다.
