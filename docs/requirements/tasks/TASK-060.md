# TASK-060 체크메이트 및 스테일메이트 판정

## 상위 Feature

- [FEATURE-001 체스 규칙 및 합법 수 판정 엔진](../features/FEATURE-001.md)

## 추천 커밋 메시지

- `feat: 체크메이트 및 스테일메이트 판정 로직 추가`

## 목적

- 합법 수를 기반으로 게임이 정상적으로 끝나는 주요 조건인 체크메이트와 무승부 조건인 스테일메이트를 정확히 판별한다.

## 완료 기준

- 현재 턴의 플레이어가 이동할 수 있는 합법 수가 전혀 없는 상태를 파악한다.
- 위 상태에서 킹이 현재 공격받고 있다면 '체크메이트'로 판정한다.
- 위 상태에서 킹이 현재 공격받지 않고 있다면 '스테일메이트(무승부)'로 판정한다.
