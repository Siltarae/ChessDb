- `feat: 수 실행 및 상태 업데이트 엔진 추가`

## 목적

- 합법 수를 실제로 적용해 보드 상태와 보조 상태를 다음 `GameState`로 전이시키는 규칙을 고정한다.

## 완료 기준

- 합법 수를 실행하면 보드 상태와 현재 턴이 다음 상태로 갱신된다.
- 캐슬링 권리, 앙파상 타깃, halfmove/fullmove 카운터 등 보조 상태가 정확히 갱신된다.
- 원본 `GameState`를 직접 수정하지 않고 새 상태를 반환한다.
