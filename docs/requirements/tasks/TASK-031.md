## 추천 커밋 메시지

- `feat: 저장소 생성 확정 추가`

## 목적

- 입력한 이름으로 저장소를 실제 생성하는 규칙을 고정한다.

## 완료 기준

- 사용자는 입력한 이름으로 저장소를 생성할 수 있다.
- 여러 저장소를 계속 추가할 수 있다.
- 저장소는 기보와 통계의 기본 집계 단위로 취급된다.
