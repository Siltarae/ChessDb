# TASK-023 새로고침 후 초안 복원

## 상위 Feature

- [FEATURE-003 기보 메타데이터 입력 및 초안 저장 관리](../features/FEATURE-003.md)

## 추천 커밋 메시지

- `feat: 새로고침 후 초안 복원 추가`

## 목적

- 브라우저 새로고침 이후에도 작업 중이던 초안을 다시 여는 규칙을 고정한다.

## 완료 기준

- 사용자가 입력 중이던 기보는 새로고침 이후에도 유지된다.
- 복원 대상에는 보드 상태, 수순 목록, 코멘트, 어노테이션이 포함된다.
- 초안 복원이 자동 저장 규칙과 연결되어 있다.

## 관련 메모

- 여러 초안을 동시에 복원하는 모델은 현재 범위에서 제외한다.
