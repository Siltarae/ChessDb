## 추천 커밋 메시지

- `feat: 반수별 착수 평가 기호 입력 추가`

## 목적

- 각 반수에 착수 평가 기호를 붙이는 규칙을 고정한다.
- UI 기준은 `docs/ui/FEATURE-003-tab-evaluation.svg`의 평가 탭 상태를 따른다.

## 완료 기준

- 각 반수마다 착수 평가 기호를 입력할 수 있다.
- 평가 기호는 반수와 직접 연결된다.
- 평가 기호는 초안 저장 범위에 포함된다.
- 평가 기호는 `!`, `!!`, `?`, `??`, `!?`, `?!` 중 하나로 제한한다.
- `없음`은 별도 저장값이 아니라 현재 반수의 평가 기호를 제거하는 액션이다.

## 관련 메모

- 일반 주석 블록이나 Variation은 현재 범위에서 제외한다.
