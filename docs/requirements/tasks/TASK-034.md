## 목적

## 완료 기준

- 저장소에 진입했을 때 FEATURE-002에서 만든 보드 중심 뷰가 가장 먼저 보여진다.
- 기본 보드뷰는 새 기보 입력 전용 화면이나 FEATURE-006 통계 분석 화면이 아니다.
- 저장 버튼 위치는 보이지만 실제 기존 기보 업데이트 저장은 별도 Task에서 다룬다.
- 원본 기준 작업 상태 복원은 별도 Task에서 다룬다.
- 기본 화면 규칙이 저장소 진입 흐름과 연결되어 있다.
