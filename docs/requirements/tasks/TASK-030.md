## 목적

- 저장소 생성 시 이름을 직접 정하는 규칙을 고정한다.

## 완료 기준

- 사용자는 저장소 이름을 직접 정할 수 있다.
- 저장소 이름은 생성 과정의 필수 정보로 취급된다.
- 저장소를 이름으로 구분하는 사용 흐름이 문서에 드러나 있다.

## 관련 메모

- 저장소 이름 수정 기능은 현재 합의 범위에 없다.
