# TASK-059 체스 보드 상태 데이터 모델 정의

## 상위 Feature

- [FEATURE-001 체스 규칙 및 합법 수 판정 엔진](../features/FEATURE-001.md)

## 추천 커밋 메시지

- `feat: 체스 보드 상태 데이터 모델 정의 (FEN 구조)`

## 목적

- 합법 수 판정의 기반이 되는 체스 보드의 현재 상태(기물 위치, 턴, 캐슬링 권한, 앙파상 위치, 반수 카운트 등)를 표현하는 데이터 구조를 정의한다.

## 완료 기준

- 체스 보드 64칸의 상태를 나타내는 자료구조(예: 1차원 또는 2차원 배열)가 구현되어야 한다.
- 현재 턴, 캐슬링 권한, 앙파상 대상 칸, 50수 규칙용 반수 카운트, 전체 수 카운트를 포함해야 한다.
- 초기 보드 상태를 나타내는 기본 FEN 파싱/생성 기능이 포함되면 좋다.
