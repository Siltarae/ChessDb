## 목적

- 수동 저장이 초안을 어떤 상태로 바꾸는지 고정한다.

## 완료 기준

- 수동 저장은 초안을 정식 기보로 확정하는 동작이다.
- 수동 저장은 새 기보를 임의로 추가 생성하는 동작이 아니다.
- `FEATURE-004` 구현 전까지 수동 저장은 시스템 기본 저장소를 대상으로 처리된다.
- 정식 저장 후 상세 뷰로 이동하는 흐름과 연결된다.

## 관련 메모

- 자동 저장은 초안 상태를 위한 기능이다.
- 시스템 기본 저장소는 `FEATURE-003` 진행을 위한 임시 전제이며, 사용자 저장소 생성/선택 흐름은 `FEATURE-004`에서 다룬다.
