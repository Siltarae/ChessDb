# TASK-058 통계 패널 표시

## 상위 Feature

- [FEATURE-006 분석 보드와 저장소 통계 조회](../features/FEATURE-006.md)

## 추천 커밋 메시지

- `feat: 통계 패널 표시 추가`

## 목적

- 집계된 후보 수와 결과 비율을 사용자에게 보여주는 화면 규칙을 고정한다.

## 완료 기준

- 통계 화면은 다음 수 후보와 결과 비율을 함께 보여준다.
- 통계 패널은 분석 보드 뷰의 핵심 출력 영역이다.
- 초기 통계 범위가 다음 수 후보와 결과 비율 조회까지만 한정되어 있다.

## 관련 메모

- 고급 필터링 UI는 후순위 범위다.
