## 목적

- 착수 결과를 수순 목록으로 누적 기록하는 규칙을 고정한다.
- UI 기준은 `docs/ui/FEATURE-002-board-interaction.svg`의 우측 수순 관리 패널과 2열 수순 목록을 따른다.

## 완료 기준

- 사용자가 둔 수는 수순 목록으로 함께 보여진다.
- 보드 상태와 수순 목록이 같은 진행 상태를 반영한다.
- 수순 목록은 백/흑을 한 행에 나누어 표시하고, 마지막 착수 셀만 강조한다.
- 첫 착수 전에는 수순 목록을 숨기거나 축약 안내만 표시한다.
- 특수 이동과 프로모션도 수순 목록에 반영될 수 있어야 한다.
