## 목적

- 실제 브라우저에서 기보 입력 화면의 최소 사용자 흐름을 검증한다.
- `TASK-016`의 완료 판정에는 포함하지 않고, `TASK-016` 이후 별도 E2E 안전망으로 둔다.

## 완료 기준

- 기보 입력 화면에 진입하면 보드가 표시된다.
- `e2` 백 폰을 선택하면 합법 수 후보가 표시된다.
- `e4`를 클릭하면 폰이 `e4`로 이동하고 현재 턴이 흑으로 전환된다.
- 불법 칸 클릭은 보드 상태와 턴을 바꾸지 않는다.
- SAN, 수순 목록, 프로모션, 체크, 게임 종료 표시는 이번 Task의 필수 단언에 포함하지 않는다.

## 관련 메모

- E2E 환경은 `TASK-099`에서 준비한다.
- 이후 `TASK-096`, `TASK-098`, `TASK-097`, `TASK-017` 완료 뒤 각 흐름의 E2E를 별도로 보강할 수 있다.
