# TASK-094 Zod를 활용한 전역 환경 변수(.env) 검증 체계 구축

## 상위 Feature

- [FEATURE-003 기보 메타데이터 입력 및 초안 저장 관리](../features/FEATURE-003.md)

## 목적

- 애플리케이션 실행에 필요한 필수 환경 변수의 유무와 타입을 런타임 시작 시점에 강제로 검증(Fail-Fast)한다.

## 완료 조건

- [ ] `packages/shared` 또는 각 앱에 `zod`를 사용한 환경 변수 스키마 정의 파일이 생성되어야 한다.
- [ ] `.env.example` 파일이 생성되어 필요한 환경 변수 목록(DATABASE_URL 등)을 명시해야 한다.
- [ ] 서버 실행 시 필수 환경 변수가 없으면 에러를 던지며 종료되는지 확인해야 한다.

## 참고 사항

- 프론트엔드(`apps/web`)의 Vite 환경 변수(`VITE_`)도 동일한 방식으로 검증할 수 있도록 설계한다.
