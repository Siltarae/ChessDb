## 추천 커밋 메시지

- `feat: 기보 결과와 종료 사유 입력 추가`

## 목적

- 정식 저장 시 기록할 결과 메타데이터 구조를 고정한다.
- UI 기준은 `docs/ui/FEATURE-003-tab-game-info.svg`의 기보 정보 탭 상태를 따른다.

## 완료 기준

- 각 기보에는 게임 결과를 사용자가 직접 입력할 수 있다.
- 결과와 종료 사유는 분리해서 기록한다.
- 결과는 정식 저장 기준 `1-0`, `0-1`, `1/2-1/2` 중 하나로 제한한다.
- 종료 사유는 체크메이트, 시간패, 기권, 무승부 네 가지로 제한한다.

## 관련 메모

- 결과 자동 판정은 현재 범위에서 다루지 않는다.
