## 추천 커밋 메시지

- `feat: 체크 상태 표시 추가`

## 목적

- 착수 후 현재 턴의 왕이 체크 상태인지 웹 UI에서 즉시 확인할 수 있게 한다.
- 마지막 착수 하이라이트와 체크 표시가 충돌하지 않도록 시각 규칙을 고정한다.

## 완료 기준

- 현재 상태가 체크이면 체크받은 왕의 칸이 명확히 표시된다.
- 체크 상태 문구 또는 배지가 현재 턴 정보와 함께 표시된다.
- 체크가 해소되면 체크 표시는 사라진다.
- 마지막 착수 표시, 선택 하이라이트, 합법 수 하이라이트와 체크 표시의 우선순위가 정해져 있다.
- Task 구현 전에 `docs/ui/TASK-098-check-status.svg`를 먼저 작성하고 화면 구조를 합의한다.

## 관련 메모

- 체크메이트 종료 표시는 `TASK-097`에서 다룬다.
- 체크 상태 계산은 shared 엔진의 체크 판정 결과를 소비한다.
