## 상위 Feature

## 추천 커밋 메시지

- `feat: 기보 입력 보드 시점 토글을 추가 (TASK-101)`

## 목적

- 사용자가 기보 입력 중 보드를 백 기준 또는 흑 기준으로 전환해 볼 수 있게 한다.
- 보드 시점 변경은 표시 순서만 바꾸며, 실제 게임 상태와 수순은 변경하지 않는다.

## 완료 기준

- 수순 목록 헤더의 보조 액션 영역에 보드 시점 전환 아이콘 버튼이 표시된다.
- 버튼을 누르면 보드 표시가 백 기준과 흑 기준 사이에서 토글된다.
- 흑 기준에서는 `h1`이 화면 첫 칸, `a8`이 화면 마지막 칸으로 표시된다.
- 뒤집힌 상태에서도 클릭, 드래그, 합법 수 하이라이트, 마지막 착수 표시, 체크 표시는 실제 `Square` 좌표 기준으로 동작한다.
- 시점 토글은 기보 입력 보드에만 적용하며 분석 보드와 기보 상세 보드는 이번 범위에서 다루지 않는다.

## 관련 메모

- 단순 CSS 회전이 아니라 보드 칸 표시 순서를 바꾸는 방식으로 구현한다.
- 버튼은 텍스트 버튼이 아니라 아이콘 버튼으로 두고, 접근성 이름은 `보드 시점 전환`으로 제공한다.
- 아이콘은 기존 아이콘 라이브러리가 있으면 그 라이브러리의 회전/전환 계열 아이콘을 우선 사용한다.
