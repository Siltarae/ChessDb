# TASK-003 현재 턴 초기화와 전환 규칙

## 상위 Feature

- [FEATURE-002 기보 입력 보드 상호작용 및 수순 관리](../features/FEATURE-002.md)

## 추천 커밋 메시지

- `feat: 현재 턴 초기화와 전환 규칙 추가`

## 목적

- 기보 입력 중 어느 색이 둘 차례인지 관리하는 규칙을 고정한다.

## 완료 기준

- 새 기보 시작 시 첫 턴은 백이다.
- 합법 수를 하나 두면 턴이 상대 색으로 전환된다.
- 되돌리기와 다시하기에서도 턴 상태가 함께 맞춰진다.

## 관련 메모

- 결과 메타데이터와는 별개의 런타임 상태다.
