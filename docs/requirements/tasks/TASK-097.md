- `feat: 착수 후 게임 종료 상태 표시 추가`

## 목적

- 착수 후 shared 엔진의 게임 결과 판정을 웹 UI에 반영한다.
- 체크메이트, 스테일메이트, 50수 규칙, 3회 반복, 기물 부족 무승부를 사용자가 인지할 수 있게 한다.

## 완료 기준

- 착수 완료 후 현재 `GameState`와 반복 이력을 기준으로 게임 종료 여부를 계산한다.
- 종료 상태이면 결과와 종료 사유가 화면에 표시된다.
- 종료 상태에서는 새 착수 입력이 막히거나 명확히 비활성 상태로 표현된다.
- 되돌리기/다시하기로 종료 이전 시점으로 이동하면 종료 표시는 현재 시점에 맞게 갱신된다.
- Task 구현 전에 `docs/ui/TASK-097-game-end-status.svg`를 먼저 작성하고 화면 구조를 합의한다.

## 관련 메모

- 정식 저장용 결과/종료 사유 입력은 `TASK-026`에서 별도로 다룬다.
- 이 Task는 자동 판정 결과를 웹 상호작용 상태에 반영하는 범위만 다룬다.
