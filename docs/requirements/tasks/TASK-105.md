## 추천 커밋 메시지

- `feat: 저장소 보드 작업 상태 복원 추가`

## 목적

- 저장소 보드에서 현재 로컬 작업 상태를 원본 기준으로 되돌리는 규칙을 고정한다.

## 완료 기준

- 기존 기보를 연 상태에서는 복원 동작이 마지막 저장된 `gameId` 상태로 돌아간다.
- 기존 기보 없이 빈 작업 상태에서는 복원 동작이 표준 시작 포지션과 빈 수순 상태로 돌아간다.
- 복원은 영구 저장된 기보를 수정하거나 삭제하지 않는다.
- 복원 전 변경 사항 손실 가능성을 사용자에게 확인할 수 있다.

## 관련 메모

- `초기화`라는 표현보다 `원본으로 복원` 또는 `작업 상태 복원` 표현을 우선 검토한다.
- 기존 기보 업데이트 저장은 [TASK-104](./TASK-104.md)에서 다룬다.
- 이 Task는 저장 버튼 동작이 아니라 로컬 작업 상태와 원본 상태의 관계를 다룬다.
