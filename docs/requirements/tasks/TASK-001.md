# TASK-001 기보 입력 뷰 기본 화면 구성

## 상위 Feature

- [FEATURE-002 기보 입력 보드 상호작용 및 수순 관리](../features/FEATURE-002.md)

## 추천 커밋 메시지

- `feat: 기보 입력 뷰 기본 화면 구성`

## 목적

- 새 기보를 입력하는 전용 화면의 기본 구성을 고정한다.

## 완료 기준

- 기보 입력 뷰가 별도 화면으로 존재한다.
- 기보 입력 뷰는 보드와 수순 목록을 함께 다룰 수 있는 구조다.
- 분석 보드 뷰와 기보 입력 뷰의 책임이 구분되어 있다.

## 관련 메모

- 분석 보드에서 새 기보 작성 버튼을 누르면 이 화면으로 이동한다.
