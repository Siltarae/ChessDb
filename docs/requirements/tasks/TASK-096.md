- `feat: 프로모션 기물 선택 UI 추가`

## 목적

- 폰이 마지막 랭크에 도달하는 착수를 둘 때 사용자가 승격 기물을 직접 선택할 수 있게 한다.
- shared 엔진의 프로모션 규칙을 웹 입력 흐름에서 실제로 소비한다.

## 완료 기준

- 프로모션 착수 시 퀸, 룩, 비숍, 나이트 중 하나를 선택할 수 있다.
- 승격 기물 선택 전에는 착수가 확정되지 않고 보드 입력 충돌이 발생하지 않는다.
- 선택한 승격 기물이 `Move.promotion`으로 보존되어 착수 실행과 후속 SAN 변환에 전달된다.
- Task 구현 전에 `docs/ui/TASK-096-promotion-piece-selection.svg`를 먼저 작성하고 화면 구조를 합의한다.

## 관련 메모

- 기본 퀸 자동 승격만으로는 기보 입력 UI의 필수 체스 규칙을 만족하지 못한다.
- 프로모션 결과의 SAN 표기는 `TASK-062`에서 다룬다.
