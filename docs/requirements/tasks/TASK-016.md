## 목적

- 합법 수를 실제로 둘 때 보드 상태를 어떻게 갱신하는지 고정한다.
- UI 기준은 `docs/ui/FEATURE-002-board-interaction.svg`의 마지막 수 표시와 선택 해제 흐름을 따른다.

## 완료 기준

- 사용자가 합법 수를 선택하면 보드 상태가 갱신된다.
- 캡처와 특수 이동 결과가 보드에 반영된다.
- 착수 완료 후 선택 상태와 합법 수 하이라이트는 정리된다.
- 마지막 착수는 보드에서 이전/도착 칸이 구분되도록 표시된다.
- 턴 전환과 수순 기록의 시작점이 된다.

## 관련 메모
