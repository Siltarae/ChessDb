# EPIC-998 기보 라이브러리 관리 확장

## 개요

이 Epic은 저장한 여러 기보를 더 편하게 관리하기 위한 후순위 기능을 정의한다.

## 상태

- 후순위 백로그
- 현재 구현 대상 아님

## 목표 초안

- 사용자가 각 기보에 제목을 붙일 수 있어야 한다.
- 사용자가 특정 기보를 즐겨찾기할 수 있어야 한다.
- 사용자가 오프닝이나 특정 수순 기준으로 기보와 통계를 필터링할 수 있어야 한다.
- 사용자가 저장소를 복사하거나 저장소 간 데이터를 붙여넣을 수 있어야 한다.
- 되돌리기 이후 새 수를 둘 때 기존 이후 수열을 Variation으로 보존할 수 있어야 한다.

## 현재 범위 메모

- 구체 Feature 및 Task는 추후 정의한다.
