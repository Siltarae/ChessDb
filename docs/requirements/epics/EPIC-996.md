# EPIC-996 PGN 입출력과 포맷 연동

## 개요

이 Epic은 기보를 PGN 형태로 변환하고, 불러오고, 외부와 주고받는 기능을 정의한다.

## 상태

- 후순위 백로그
- 현재 구현 대상 아님

## 목표 초안

- 저장된 기보를 PGN 형태로 변환할 수 있어야 한다.
- PGN 텍스트를 불러와 기보로 다룰 수 있어야 한다.
- 내부 기보 모델과 PGN 표현 사이의 변환 규칙을 정할 수 있어야 한다.

## 현재 범위 메모

- 내보내기, 불러오기, 파싱, 직렬화 범위는 추후 구체화한다.
- Phase 2 이후 데이터 모델이 안정되면 다시 상세화한다.
