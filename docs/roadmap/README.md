# 추후 계획 공용 문서

## 목적

- 현재 범위에 포함하지 않은 아이디어와 후순위 계획을 한곳에서 관리한다.
- Epic, Feature, Task 문서에 흩어진 추후 계획을 모아 확인할 수 있게 한다.
- Phase 1부터 Phase 6까지 공통으로 참조하는 백로그 문서로 사용한다.

## 기록 원칙

- 현재 구현 범위와 추후 계획을 반드시 구분해서 적는다.
- 아직 확정되지 않은 아이디어는 초안으로 표시한다.
- 이미 별도 Epic으로 분리된 내용은 해당 문서 링크를 함께 남긴다.
- 구체 구현 규칙보다 방향성과 분리 이유를 우선 기록한다.
- 후순위 Epic은 이 문서에서 링크와 짧은 메모로 모아 보고, 상세 내용은 각 Epic 문서에서 유지한다.

## 현재 추후 계획

### 제품 범위 확장

- [EPIC-996 PGN 입출력과 포맷 연동](../requirements/epics/EPIC-996.md)
  - PGN 변환, 불러오기, 내보내기를 위한 후순위 Epic
- [EPIC-997 사용자 인증 및 계정 시스템](../requirements/epics/EPIC-997.md)
  - 로그인과 사용자별 데이터 분리를 위한 후순위 Epic
- [EPIC-998 기보 라이브러리 관리 확장](../requirements/epics/EPIC-998.md)
  - 제목, 즐겨찾기, 고급 필터링, 저장소 복사/붙여넣기, Variation 보존 같은 확장 기능
- [EPIC-999 중간 포지션 기반 기보 입력](../requirements/epics/EPIC-999.md)
  - FEN 기반 시작 입력을 위한 후순위 Epic

### 기능 확장 메모

- Variation 보존은 현재 범위 밖에서 관리한다.
- 저장소 복사/붙여넣기는 현재 범위 밖에서 관리한다.
- 고급 필터링은 현재 범위 밖에서 관리한다.
