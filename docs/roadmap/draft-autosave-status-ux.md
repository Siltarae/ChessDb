# 초안 자동 저장 상태 UX 개선 메모

## 배경

- 현재는 착수할 때마다 `초안 저장됨` 토스트가 갱신된다.
- 연속으로 수를 두면 토스트가 사라지지 않고 계속 유지되는 느낌을 줄 수 있다.

## 추후 탐색할 방향

- 자동 저장 상태를 토스트가 아니라 더 자연스러운 상태 표시로 표현할 방법을 찾는다.
- 사용자가 실제 저장 동작을 방해받지 않으면서도 초안이 저장되고 있음을 이해할 수 있어야 한다.
- `초안 복원됨`, `초안 저장됨`, 향후 정식 저장 상태가 서로 헷갈리지 않도록 표현 체계를 분리한다.

## 현재 범위 밖인 이유

- `TASK-023`의 핵심 범위는 새로고침 후 초안 복원이다.
- 자동 저장 상태 표시 UX 개선은 별도 화면/상태 설계가 필요한 후순위 작업으로 분리한다.
