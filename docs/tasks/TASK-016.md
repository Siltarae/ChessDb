## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 기물을 선택하고 어디로 갈 수 있는지는 보여주지만, 실제로 말을 움직일 수는 없습니다.
- **이 작업의 책임**: 하이라이트된 합법 수 위치를 클릭했을 때, 실제로 착수를 수행하고 보드 상태(기물 위치, 턴 전환 등)를 갱신합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 기물을 목표 칸으로 이동시키면 보드가 즉시 업데이트되고 다음 플레이어의 턴으로 넘어갑니다.

## 🛠️ 3. 상세 기술 사양

- **로직 흐름**:
  1. 이미 기물이 선택된 상태에서 '합법 수' 칸을 클릭.
  2. `shared` 패키지의 `makeMove(gameState, from, to)` 호출.
  3. 반환된 새로운 `GameState`로 Zustand 스토어 업데이트.
  4. `selectedSquare`를 `null`로 초기화.

## ✅ 4. 완료 판정 체크리스트

- [ ] 기물이 목표 위치로 정상적으로 이동한다.
- [ ] 착수 후 현재 턴이 자동으로 전환된다.
- [ ] 캡처(상대 기물 잡기) 발생 시 상대 기물이 보드에서 제거된다.
