# 📋 개별 작업 지침서: 기존 초안 폐기 후 새 기보 시작 (TASK-024)

**작업 상태**: 대기 중  
**선행 작업**: `[TASK-022]`  
**후속 작업**: `[TASK-025]`

---

## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 초안이 자동으로 복원되지만, 사용자가 새로운 대국을 기록하고 싶을 때 기존 내용을 비우는 기능이 없습니다.
- **이 작업의 책임**: 사용자가 '새 게임' 버튼을 눌렀을 때, 사용자 확인(Confirm)을 거쳐 기존 초안 데이터를 삭제하고 초기 상태로 되돌리는 기능을 구현합니다.

## 🛠️ 3. 상세 기술 사양

- **로직**:
  1. `window.confirm` 또는 `shadcn/ui`의 AlertDialog를 통해 확인 절차 수행.
  2. 수락 시 Zustand 스토어의 `reset` 액션 호출 및 `localStorage.removeItem` 실행.

## ✅ 4. 완료 판정 체크리스트

- [ ] 사용자 확인 후 데이터가 완벽히 초기화된다.
- [ ] 초기화 후 로컬 스토리지에서도 초안 데이터가 제거됨을 확인했다.
