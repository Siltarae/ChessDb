## 🛠️ 3. 상세 기술 사양

- **데이터 필드**:
  - `result`: '1-0', '0-1', '1/2-1/2', '*'
  - `termination`: 'Checkmate', 'Timeout', 'Resignation', 'Draw'
  - `playedAt`: ISO Date String
- **UI**: `shadcn/ui`의 Select 및 DatePicker 컴포넌트 활용.
- **위치**: 기보 입력 페이지의 사이드바 또는 하단 메타데이터 패널.

## ✅ 4. 완료 판정 체크리스트

- [ ] 결과와 종료 사유가 상호 배타적으로(또는 적절히 조합되어) 입력된다.
- [ ] 선택한 날짜가 `GameState`의 메타데이터에 정확히 반영된다.
