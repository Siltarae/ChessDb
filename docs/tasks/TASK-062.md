## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 보드 상에서 기물은 움직이지만, 이를 텍스트 형태의 기보(Nf3, e4 등)로 표현할 방법이 없습니다.
- **이 작업의 책임**: 착수 정보(`from`, `to`, `captured` 등)를 체스 표준 기보법(SAN) 문자열로 변환하는 유틸리티를 작성합니다.

## 🛠️ 3. 상세 기술 사양

- **위치**: `packages/shared/src/utils/san.ts` (공통 로직이므로 shared에 위치)
- **변환 규칙**:
  - 폰 이동: `e4`, `exd5`
  - 기물 이동: `Nf3`, `Bxe4`
  - 체크: `+`, 체크메이트: `#`
  - 캐슬링: `O-O`, `O-O-O`
- **주의**: 동일한 기물이 같은 칸으로 갈 수 있는 경우의 모호성 해결(`Nbd2` 등) 로직 포함.

## ✅ 4. 완료 판정 체크리스트

- [ ] `shared` 패키지의 단위 테스트를 통해 변환 결과가 검증되었다.
