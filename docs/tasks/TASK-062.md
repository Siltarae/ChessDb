## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 기물 이동 로직은 완성되었으나, 이를 사람이 읽기 좋은 `Nf3`, `e4`, `O-O` 같은 표준 기보(SAN)로 표현하는 기능은 없습니다.
- **이 작업의 책임**: 착수 정보(`Move`)와 실행 전/후의 `GameState`를 입력받아, 체스 표준 기보법(SAN) 문자열을 생성합니다.
- **경계 메모**: 동일한 기물이 같은 칸으로 이동할 수 있는 경우의 모호성 해결(예: `Nbd2`, `N1d2`) 로직을 반드시 포함해야 합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 모든 착수 상황을 국제 표준 기보법(SAN) 문자열로 정확히 변환하여 기보 저장 및 출력에 활용합니다.
- **이번 작업의 최소 결과물**:
  - `packages/shared/src/utils/san-converter.ts` 생성.
  - `packages/shared/src/utils/san-converter.spec.ts` 테스트 통과.
- **성공 기준 (AC)**:
  - 일반 이동(`e4`, `Nf3`), 캡처(`exd5`, `Bxe4`), 캐슬링(`O-O`, `O-O-O`)을 정확히 변환한다.
  - 체크(`+`) 및 체크메이트(`#`) 기호를 올바르게 추가한다.
  - 기물 이동의 모호성을 해결하기 위한 파일/랭크 표기 로직을 갖춘다.

## 🛠️ 3. 상세 기술 사양

- **핵심 함수 시그니처**:

  ```ts
  import { GameState, Move, Square } from '../models/game-state.js';

  export function convertToSAN(state: GameState, move: Move): string;
  ```

- **SAN 변환 규칙**:
  1. **기물 표시**: 킹(K), 퀸(Q), 룩(R), 비숍(B), 나이트(N). 폰은 표시 생략.
  2. **모호성 해결**: 같은 타입의 다른 기물이 같은 칸으로 갈 수 있다면 `from`의 파일(예: `Nbd2`), 그래도 같으면 랭크(예: `N1d2`), 둘 다 필요하면 파일+랭크(예: `Nb1d2`) 추가.
  3. **캡처**: `x` 기호 추가. 폰 캡처 시 시작 파일 명시 (예: `exd5`).
  4. **캐슬링**: 킹사이드 `O-O`, 퀸사이드 `O-O-O`.
  5. **승진**: 이동 뒤에 `=Q` 등의 형식 추가.
  6. **상태 기호**: 체크 시 `+`, 체크메이트 시 `#`.
- **권장 네이밍**:
  - `convertToSAN`: 메인 변환 함수.
  - `getPieceNotation`: 기물 약어 반환 내부 헬퍼.
  - `resolveAmbiguity`: 모호성 해결용 파일/랭크 식별 헬퍼.
- **필수 describe/it 목록**:
  - describe: `convertToSAN`
    - it: `폰이 빈 칸으로 전진할 때 e4 형식을 반환해야 한다`
    - it: `나이트가 이동할 때 Nf3 형식을 반환해야 한다`
    - it: `폰이 기물을 잡을 때 exd5 형식을 반환해야 한다`
    - it: `두 나이트가 같은 칸으로 갈 수 있을 때 Nbd2 형식을 반환해야 한다`
    - it: `체크메이트 상황일 때 # 기호로 끝나야 한다`
- **최소 테스트 개수**: 10개 이상 (다양한 이동 및 캡처 케이스)

## ⚖️ 4. 기술 제약 및 규칙

- **상태 의존성**: 모호성 해결 및 체크 여부 판단을 위해 반드시 이동 전의 `GameState` 전체가 필요합니다.
- **순수성**: 입력을 변경하지 않고 문자열만 반환합니다.

## 🧪 5. 검증 시나리오 및 단언

1. **정상 시나리오: 폰 캡처**
   - Given: 백 폰 e4, 흑 폰 d5 상황.
   - When: `convertToSAN(state, { from: e4, to: d5 })`
   - Then: `exd5`.

2. **경계 시나리오: 모호한 나이트**
   - Given: b1과 f1에 백 나이트, d2가 비어있음.
   - When: b1 나이트가 d2로 이동.
   - Then: `Nbd2`.
