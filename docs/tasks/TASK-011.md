## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 각 기물의 '의사 합법 수(Pseudo-legal moves)'와 특정 위치가 공격받고 있는지 여부(`isCheck`)는 구현되어 있습니다.
- **이 작업의 책임**: 의사 합법 수 중 '자신의 킹을 체크 상태로 만드는 수'를 제외하여 최종적인 '진짜 합법 수'를 필터링합니다.

## 🎯 1. 작업 목표

- **최종 상태**: `getLegalMoves(square, state)` 함수가 킹의 안전을 보장하는 수만 반환합니다.

## 🛠️ 3. 상세 기술 사양

- **핵심 조립/흐름 규칙**:
  1. 특정 위치(`square`) 기물의 모든 의사 합법 수를 가져옵니다.
  2. 각 수에 대해 '가상으로 착수'한 후의 보드 상태(`nextState`)를 생성합니다.
  3. `isCheck(nextState, currentTurn)`를 실행하여 자신의 킹이 체크 상태인지 확인합니다.
  4. 체크 상태라면 해당 수는 합법 수 목록에서 제외합니다.
- **공개 함수 시그니처**:
  ```ts
  export function getLegalMoves(square: Square, state: GameState): Square[];
  export function makeMove(state: GameState, from: Square, to: Square): GameState; // 가상 착수용
  ```

## ✅ 4. 완료 판정 체크리스트

- [ ] 핀(Pin)에 걸린 기물이 킹을 위험에 빠뜨리며 이동하는 것을 차단한다.
- [ ] 체크 상태일 때, 체크를 해소하지 못하는 모든 수를 제외한다.
- [ ] 킹 스스로가 공격받는 칸으로 이동하는 것을 차단한다.
