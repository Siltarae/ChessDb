## 🛠️ 3. 상세 기술 사양

- **핵심 조립/흐름 규칙**:
  - 나이트는 'L'자 모양으로 점프하며, 경로상의 기물 유무와 상관없이 목적지에 자신의 기물만 없으면 이동 가능합니다.

- [ ] 8방향 'L'자 이동이 모든 위치에서 정확히 계산된다.
