# 📋 개별 작업 지침서: 킹(King) 기본 이동 합법 수 판정 (TASK-009)

**작업 상태**: 대기 중  
**선행 작업**: `[TASK-059]`  
**후속 작업**: `[TASK-010]` (체크 판정)

---

## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 모든 기물의 기본 이동이 구현되었으며, 마지막으로 킹의 기본 1칸 이동을 구현할 차례입니다.
- **이 작업의 책임**: 킹의 8방향 1칸 이동 로직을 구현합니다. (캐슬링은 별도 태스크에서 처리)

## 🎯 1. 작업 목표

- **최종 상태**: 킹이 주변 8칸 중 아군 기물이 없고 보드 내부에 있는 위치를 반환합니다.

## 🛠️ 3. 상세 기술 사양

- **핵심 규칙**:
  - 8방향 오프셋: 나이트와 비슷하게 점프가 아닌 1칸 이동으로 처리.
  - **주의**: 이 단계에서는 킹이 이동하려는 칸이 공격받고 있는지(체크 여부)는 고려하지 않습니다. 이는 `TASK-011`에서 필터링으로 처리합니다.

## ✅ 4. 완료 판정 체크리스트

- [ ] 주변 8칸 이동이 정확히 계산된다.
- [ ] 아군 기물이 있는 칸으로의 이동이 차단된다.
- [ ] 보드 경계를 벗어나는 이동이 제외된다.
