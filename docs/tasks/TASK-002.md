## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 레이아웃만 존재하며 보드 내부에는 아무것도 그려지지 않은 상태입니다.
- **이 작업의 책임**: `@chess-db/shared`에서 정의한 `GameState`를 기반으로 8x8 체스 보드와 초기 기물 배치를 렌더링합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 화면에서 표준 체스 시작 배치를 확인합니다.

## 🛠️ 3. 상세 기술 사양

- **컴포넌트 구조**: `Board` -> `Square` -> `Piece`
- **렌더링 규칙**:
  - `shared` 패키지의 `INITIAL_BOARD` (또는 초기화 로직)에서 데이터를 가져옵니다.
  - a1이 좌측 하단, h8이 우측 상단이 되도록 행/열을 매핑합니다.
- **기물 이미지**: SVG 또는 폰트 아이콘(Lucide 등)을 사용하여 기물 타입과 색상에 맞게 표시합니다.

## ✅ 4. 완료 판정 체크리스트
