## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 각 기물별로 '내가 어디로 갈 수 있는지'는 알지만, '상대가 우리 킹을 공격하고 있는지'는 모르는 상태입니다.
- **이 작업의 책임**: 현재 턴의 킹이 상대방 기물에 의해 공격받고 있는지를 판정하는 핵심 함수를 구현합니다.

## 🎯 1. 작업 목표

- **최종 상태**: `isCheck(state, color)` 함수를 통해 특정 색상의 킹이 위협받고 있는지 여부를 `boolean`으로 반환합니다.

## 🛠️ 3. 상세 기술 사양

- **공개 함수 시그니처**:
  ```ts
  export function isSquareAttacked(square: Square, attackerColor: Color, state: GameState): boolean;
  export function isCheck(state: GameState, color: Color): boolean;
  ```
- **핵심 조립 규칙 (isSquareAttacked)**:
  - 특정 칸(`square`)이 특정 색상(`attackerColor`)의 기물에 의해 공격받고 있는지 확인하려면:
    - 해당 칸을 기준으로 역으로 검사: "내가 만약 룩이라면 이 방향에 상대 룩이나 퀸이 있는가?", "내가 나이트라면 이 위치에 상대 나이트가 있는가?"
    - 또는 모든 상대 기물의 합법 수를 계산하여 해당 칸이 포함되는지 확인 (성능상 역방향 검사 권장).

## ✅ 4. 완료 판정 체크리스트

- [ ] 킹의 현재 위치를 정확히 파악한다.
- [ ] 모든 종류의 기물(폰, 나이트, 슬라이딩 기물)에 의한 공격을 감지한다.
- [ ] 자신의 킹이 체크 상태인지 여부를 정확히 반환한다.
