## 🎯 1. 작업 목표

- **최종 상태**: `getGameResult(state)` 함수가 게임 진행 중, 체크메이트, 스테일메이트 상태를 정확히 반환합니다.

## 🛠️ 3. 상세 기술 사양

- **핵심 판정 규칙**:
  1. 현재 플레이어의 모든 기물에 대해 `getLegalMoves`를 합산합니다.

     - `isCheck(state, turn)`이 `true`이면 ➔ **체크메이트 (상대 승리)**
