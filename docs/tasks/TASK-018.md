## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 수순 이력은 쌓이지만, 과거 시점으로 돌아가는 기능은 없습니다.
- **이 작업의 책임**: 수순 목록에서 이전 수를 선택하거나 전용 버튼을 눌러 보드 상태를 과거/미래로 이동시키는 기능을 구현합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 '되돌리기'를 누르면 보드가 한 수 전으로 돌아가고, 새로운 수를 두면 기존 이후 수순은 삭제됩니다.

## 🛠️ 3. 상세 기술 사양

- **되돌리기(Undo)**: `history` 배열의 마지막 요소를 꺼내고(`pop`), 해당 수 직전의 `stateBefore`를 현재 `gameState`로 복구.
- **다시하기(Redo)**: 별도의 `redoStack`을 관리하거나 `history` 포인터를 이동하여 구현.
- **분기 처리**: 되돌리기 상태에서 새 수를 두면 `redoStack`을 비움.

## ✅ 4. 완료 판정 체크리스트

- [ ] 되돌리기 시 보드 상태와 턴이 이전 상태로 완벽히 복구된다.
- [ ] 수순 목록에서 특정 수를 클릭했을 때 해당 시점의 보드가 표시된다.
- [ ] 새로운 수 착수 시 미래 수순이 적절히 삭제(Truncate)된다.
