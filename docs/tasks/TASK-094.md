## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: `.env` 파일을 통해 설정을 관리하기 시작했으나, 필수 변수 누락 시 서버가 실행 중에 에러를 내는 불안정한 상태입니다.
- **이 작업의 책임**: 애플리케이션 시작 시점에 모든 필수 환경 변수(`DATABASE_URL` 등)를 Zod 스키마로 강제 검증하여, 설정 오류 시 즉시 실행을 중단(Fail-Fast)시킵니다.

- **이번 작업에서 하지 않는 것**: `[TASK-090]` (DTO 연결)에 연결된 후속 책임은 이번 태스크에서 함께 닫지 않는다.

- **경계 메모**:
  - 이번 태스크는 Zod를 활용한 전역 환경 변수 검증 체계 구축 범위만 닫고, 후속 태스크 또는 인접 Feature의 세부 구현은 여기서 함께 처리하지 않는다.

## 🛠️ 3. 상세 기술 사양

- **위치**: `apps/api/src/core/config/env.config.ts`
- **구현 방식**:
  ```ts
  const envSchema = z.object({
    DATABASE_URL: z.string().url(),
    PORT: z.coerce.number().default(3000),
    NODE_ENV: z.enum(['development', 'production', 'test']).default('development'),
  });
  export const env = envSchema.parse(process.env);
  ```
- **NestJS 연동**: `ConfigModule.forRoot({ validate: (config) => envSchema.parse(config) })`와 같이 연동 검토.
- **필수 describe/it 목록**:
  - describe: `Environment Variable Validation`
    - it: `필수 변수(DATABASE_URL) 누락 시 프로세스가 에러를 던지며 중단되어야 한다`
    - it: `유효한 변수가 주어지면 파싱된 설정 객체를 반환해야 한다`

- **핵심 조립/흐름 규칙**:
  - 서버 시작 시 Zod 검증 로직이 가장 먼저 실행됨을 확인했다.
  - `.env.example` 파일에 현재 필요한 모든 변수 목록이 최신화되었다.
  - 잘못된 형식의 URL이나 숫자 값이 들어올 경우 정확한 검증 에러를 출력한다.

- **데이터 모델 해석**:
  - 핵심 데이터는 필수 환경 변수 이름, 포맷 규칙, 파싱 결과 타입입니다.
  - 입력은 `process.env`이고 출력은 검증을 통과한 설정 객체입니다.

- **외부 의존성**:
  - `zod`
  - `@nestjs/config` 또는 동등 설정 모듈

- **import/export 규칙**:
  - `../architecture/project-rules.md`의 named export, 상대 경로 최소화, `.js` 확장자 규칙을 따른다.

- **권장 네이밍**:
  - 공개 함수/타입 이름: Zod를 활용한 전역 환경 변수 검증 체계 구축 책임이 드러나는 이름
  - 내부 helper 이름: 역할이 바로 드러나는 동사형 또는 조합형 이름
  - 핵심 변수명: 상태와 대상이 분명한 이름
  - 피해야 할 이름: data, item, obj, temp

- **이름별 사용 의도와 적용 시점**:
  - `envSchema`, `env`, `parseEnv`는 환경 변수 검증 레이어 이름으로 사용합니다.
  - `DATABASE_URL`, `PORT`, `NODE_ENV`는 필수 키 이름으로 유지합니다.

- **인수 이름 가이드**:
  - `config`, `parsedEnv`, `envSchema`

- **짧은 예시 골격**:

```ts
const env = envSchema.parse(process.env);
```

- **최소 테스트 개수**:
  - 최소 3개

- **반드시 포함할 실패 시나리오**:
  - 앱 시작 뒤에야 환경 변수 누락을 알게 되는 경우
  - `DATABASE_URL` 같은 필수 URL 형식을 문자열로만 통과시키는 경우
