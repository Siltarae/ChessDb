## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 분석 보드와 기보 상세 뷰가 각각 존재하지만, 특정 기보를 보다가 해당 시점부터 분석을 이어가는 연결 고리가 없습니다.
- **이 작업의 책임**: 기보 상세 화면에서 '이 시점부터 분석하기' 버튼을 제공하여, 해당 포지션 정보를 분석 보드로 그대로 넘겨주는 기능을 구현합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 과거 대국의 특정 수순에서 분석 보드로 즉시 이동하여 다른 수 후보를 탐색할 수 있습니다.

## 🛠️ 3. 상세 기술 사양

- **데이터 전달**: 현재 상세 뷰의 `GameState`를 `AnalysisStore`의 초기 상태로 주입하며 페이지 이동.
- **UI**: 기보 상세 화면의 컨트롤바 또는 수순 목록 하단에 '분석 보드에서 열기' 버튼 배치.

## ✅ 4. 완료 판정 체크리스트

- [ ] 버튼 클릭 시 분석 보드로 페이지가 전환되며 보드 배치가 유지된다.
- [ ] 분석 보드 진입 후 해당 시점부터 새로운 수를 두어 통계를 조회할 수 있다.
