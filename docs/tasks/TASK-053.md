## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 분석 보드와 기보 상세 화면이 각각 독립적으로 작동하고 있습니다. 특정 기보를 복기하다가 궁금한 시점부터 분석을 시작할 수 있는 연결 고리가 필요합니다.
- **이 작업의 책임**: 기보 상세 화면에서 특정 수순의 포지션을 분석 보드로 전송하고 페이지를 전환하는 기능을 구현합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 과거 대국의 특정 수순에서 '분석 보드에서 열기'를 누르면, 해당 시점의 기물 배치 그대로 분석 도구가 열려 다양한 후보 수의 승률을 확인할 수 있습니다.

## 🛠️ 3. 상세 기술 사양

- **데이터 연동**:
  - 상세 뷰에서 현재 보고 있는 `GameState`를 `AnalysisStore`의 `setInitialState` 액션으로 전달.
  - `react-router`의 `useNavigate`를 사용하여 분석 보드 URL로 이동.
- **UI**:
  - 기보 상세 페이지 하단 컨트롤 바에 '이 포지션 분석하기' 아이콘/버튼 추가.

## ✅ 7. 완료 판정 체크리스트

- [ ] 버튼 클릭 시 분석 보드 페이지로 즉시 전환된다.
- [ ] 상세 뷰에서 보고 있던 보드 상태(기물 위치, 턴 등)가 분석 보드에 완벽히 복제된다.
- [ ] 분석 보드 진입 후 해당 시점부터 새로운 수를 두어볼 수 있다.
