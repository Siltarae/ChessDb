## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 보드 표시와 기물 배치까지 가능하지만, 사용자의 클릭에 반응하는 상호작용은 없는 상태입니다.
- **이 작업의 책임**: 사용자가 기물을 클릭했을 때 해당 칸을 강조하고, 이동 가능한 합법 수 위치에 시각적 가이드를 표시합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 기물을 클릭하면 선택된 칸이 하이라이트되고, 갈 수 있는 모든 칸에 작은 점(Dot)이나 반투명 효과가 나타납니다.

## 🛠️ 3. 상세 기술 사양

- **상태 관리**:
  - `selectedSquare: Square | null` 상태를 스토어에 추가.
- **인터랙션 흐름**:
  1. 기물이 있는 칸 클릭 시 `selectedSquare` 업데이트.
  2. 이미 선택된 칸을 다시 클릭하면 선택 해제.
  3. 선택된 상태에서 `shared` 엔진의 `getLegalMoves(selectedSquare, gameState)` 호출.
  4. 반환된 좌표 배열을 활용하여 `Square` 컴포넌트에 `isHighlighted` prop 전달.
- **UI**:
  - 선택된 칸: 테두리 강조 또는 배경색 변경.
  - 합법 수 칸: 중앙에 작은 원(Circle) 오버레이 표시.

## ✅ 7. 완료 판정 체크리스트

- [ ] 아군 기물 클릭 시 해당 칸과 이동 가능 칸이 즉시 하이라이트된다.
- [ ] 빈 칸이나 상대 기물(선택 전) 클릭 시에는 하이라이트가 발생하지 않는다.
- [ ] 다른 기물을 클릭하면 이전 하이라이트가 사라지고 새로운 기물 기준으로 갱신된다.
