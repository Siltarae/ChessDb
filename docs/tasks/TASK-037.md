## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 저장소에 진입하면 분석 보드만 보이며, 해당 저장소에 저장된 기보들을 확인할 수 있는 화면이 없습니다.
- **이 작업의 책임**: 특정 저장소 내에 저장된 모든 기보를 날짜순으로 조회하여 목록으로 보여줍니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 저장소 내 '기보 목록' 탭을 클릭하면 저장된 기보들의 날짜, 결과(1-0 등), 종료 사유를 한눈에 확인합니다.
- **핵심 산출물**: `GameListPage.tsx`, `GameListItem.tsx`, `GET /api/repositories/:id/games` API.

## 🛠️ 3. 상세 기술 사양

- **데이터 모델 (GameSummary)**: `{ id: string, playedAt: string, result: string, firstMoves: string[] }`
- **정렬**: `playedAt` 기준 내림차순 (최신순).
- **UI**: 기보의 핵심 정보와 함께, 초반 몇 수를 미리보기(SAN 리스트)로 보여주어 식별을 돕습니다.

## ✅ 4. 완료 판정 체크리스트

- [ ] 저장된 기보들이 리스트 형태로 정상 출력된다.
- [ ] 대국 날짜와 결과 정보가 정확히 표시된다.
- [ ] 목록에서 특정 기보를 선택할 수 있는 구조가 마련되었다.
