## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 분석 보드 레이아웃과 통계 API는 준비되었으나, 사용자에게 데이터를 표나 그래프로 보여줄 시각적 요소가 부족합니다.
- **이 작업의 책임**: 분석 보드 우측 패널에 API 결과를 렌더링하여, 사용자가 현재 포지션에서의 수 후보와 과거 성적을 한눈에 파악하게 합니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 보드에서 수를 두면 우측 패널이 즉시 업데이트되며, 가장 많이 두어진 수 순서대로 리스트가 나타나고 각 수별 승률이 막대 그래프로 표시됩니다.

## 🛠️ 3. 상세 기술 사양

- **데이터 연동**:
  - `useAnalysisStore`에서 관리하는 `currentFen`이 바뀔 때마다 `TanStack Query`가 자동으로 통계 API를 재요청.
- **UI 컴포넌트**:
  - `StatsTable`: `shadcn/ui`의 Table을 사용하여 수(SAN), 대국 수(Count), 승률 표시.
  - `WinRateBar`: 가로형 스택 막대 그래프. 백승(White), 무승부(Gray), 흑승(Black) 영역의 너비를 승률 비율에 맞게 렌더링.
- **UX**:
  - 통계 리스트의 특정 수 클릭 시 해당 수를 실제로 분석 보드에 두어보는 '미리보기' 기능 연동.
