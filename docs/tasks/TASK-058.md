# 📋 개별 작업 지침서: 통계 패널 표시 (TASK-058)

**작업 상태**: 대기 중  
**선행 작업**: `[TASK-054]` (통계 API)  
**후속 작업**: `FEATURE-006` 완료

---

## 0. 현재 코드 상태와 이 작업의 위치

- **현재 상태 요약**: 분석 보드 레이아웃과 통계 API는 준비되었으나, 사용자에게 데이터를 시각적으로 보여줄 패키지가 없습니다.
- **이 작업의 책임**: 분석 보드 우측 패널에 API 결과를 표나 차트 형태로 렌더링하여 사용자에게 보여줍니다.

## 🎯 1. 작업 목표

- **최종 상태**: 사용자가 보드에서 수를 두면 우측 패널에 실시간으로 "이 포지션에서 가장 많이 두어진 수"와 "승률"이 나타납니다.

## 🛠️ 3. 상세 기술 사양

- **데이터 연동**: `TanStack Query`의 `useQuery`를 사용하여 보드의 FEN이 변경될 때마다 통계 API 호출.
- **시각화**: 
  - 수 후보 목록: SAN 기보법과 함께 횟수 표시.
  - 승률 바(Win Rate Bar): 백승(백색), 무승부(회색), 흑승(검은색) 비율을 가로 막대 그래프로 시각화.
- **UI**: `shadcn/ui`의 Table 또는 전용 시각화 컴포넌트 활용.

## ✅ 4. 완료 판정 체크리스트

- [ ] 보드 상태가 바뀔 때마다 통계 패널이 자동으로 업데이트된다.
- [ ] 각 수별 승률 통계가 가독성 있게 표시된다.
- [ ] 통계가 없는 새로운 포지션일 경우 "데이터가 없습니다" 메시지가 출력된다.
